# Marine_Monopolist

Developed with Unreal Engine 5

Игра рыбалка в скандинавской мифологии, в жанре айдлер кликер 3D.

Игровой процесс:
Игрок играет за рыбака и должен ловить рыбу. Сам процесс рыбалки автоматический, но игрок может кликать, чтобы быстрей добывать рыбу. Когда игрок ловит рыбу, то он получает игровую валюту в соотвествии от редкости рыбы. За валюту игрок может прокачаться, например купить сети или улучшить свою удочку и плот, что увеличивает количество дохода с рыбалки. Днём игрок ловит обычную рыбу, а ночью мифическую, что приносит ему больше денег.
